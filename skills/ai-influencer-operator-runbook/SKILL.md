# AI Influencer Daily Operator Runbook

Use this skill when the user needs a structured, step-by-step daily workflow for operating an AI influencer network. This runbook ties together content generation, distribution, and monetization into a daily checklist.

## 1. Morning Routine: Engagement & Distribution (1 Hour)

The first task of the day is pushing content to the top-of-funnel platforms to drive traffic.

1.  **Check Proxies & Antidetect:** Open AdsPower (or similar antidetect browser). Ensure the proxy for Character A (e.g., Mei Lin) is active and clean.
2.  **Instagram/TikTok Posting (Batch 1):** Post the morning Reel/TikTok (prepared the previous day). Use a trending audio and a POV text overlay.
3.  **Reddit Distribution:** Post 1-2 static images to highly targeted, niche subreddits. Use native, non-promotional titles.
4.  **Twitter (X) Engagement:** Post a "tease" image. Spend 15 minutes engaging with the RT (Retweet) Pod/Network. Reply to 3 viral tweets with character-relevant content.
5.  **Telegram Broadcast:** Send the daily "Good Morning" voice note or blurred teaser to the free Telegram channel.
6.  **Repeat for Character B:** Switch to Character B's profile in the antidetect browser and repeat steps 2-5.

## 2. Mid-Day Routine: Content Generation Pipeline (2-3 Hours)

Generate the content required for the *next* 48 hours to maintain the 3-5 posts/day volume.

1.  **Concept Ideation:** Use the `ai-influencer-content-generation` skill to generate 10 new prompts based on the "Playful Tease" formula and "Plausible Deniability" hooks.
2.  **Base Generation (ComfyUI / Midjourney):** Generate the base images. Ensure lighting is flattering and skin is flawless.
3.  **Face Locking (Roop / IP-Adapter):** Run the base images through the face-swap pipeline to lock the character's identity (e.g., Mei Lin's face).
4.  **Video Generation (Wan 2.1 / Kling):** Select the best 3-5 images and run them through the image-to-video pipeline to create Reels/TikToks.
5.  **Upscaling & Refinement:** Upscale the final assets and add slight film grain to remove the "AI plastic" look.
6.  **Vault Production:** Generate 2-3 highly suggestive/explicit assets specifically for the Fanvue Chat Vault.

## 3. Afternoon Routine: Scheduling & Organization (1 Hour)

Prepare the generated content for distribution.

1.  **Caption Writing:** Use the `ai-influencer-distribution` skill to write POV captions and identify trending hashtags for the generated videos.
2.  **Scheduling (Later.com / Metricool):** Schedule the 3-5 Reels/TikToks for the next day, spacing them 3-4 hours apart.
3.  **Vault Organization:** Upload the newly generated Fanvue assets to the Infloww CRM. Tag them by suggestiveness level (SFW, Soft Pitch, Explicit) so chatters can easily find them.

## 4. Evening Routine: Monetization & Analytics (1 Hour)

Review revenue and manage the chat team.

1.  **Chatter Check-In:** Review the Infloww CRM. Check the conversion rates of the human chatters. Are they following the Escalation Ladder?
2.  **Whale Management:** Personally review the chats of the top 5 spenders ("whales"). Ensure they are receiving high-quality GFE (Girlfriend Experience).
3.  **Analytics Review:**
    *   *Traffic:* Which Reddit posts drove the most clicks to Bouncy.ai?
    *   *Growth:* Which Instagram Reel had the highest watch time?
    *   *Revenue:* What was the PPV unlock rate on Fanvue today?
4.  **Adjust Strategy:** If a specific outfit or hook performed well today, add it to tomorrow's Content Generation Pipeline.

## 5. Output Generation

When the user asks for a daily schedule, workflow optimization, or troubleshooting for their operations, use this runbook to provide a structured, step-by-step checklist. Emphasize batch production and strict separation of characters via antidetect browsers.
